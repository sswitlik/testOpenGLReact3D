Detailed Description

This class represents a mesh made of triangles.

A TriangleMesh contains one or several parts. Each part is a set of triangles represented in a TriangleVertexArray object describing all the triangles vertices of the part. A TriangleMesh object is used to create a ConcaveMeshShape from a triangle mesh for instance.
